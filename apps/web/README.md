# BorderVet

MVP freelance para farmacia veterinaria con catalogo, carrito y pedido por WhatsApp.
Dirigido a clientes en Tijuana que compran productos o solicitan asesoria virtual.

## Stack
- Astro
- TypeScript
- Tailwind CSS
- Swiper
- GSAP
- Lenis
- Netlify

## Funcionalidades principales
- Home comercial con hero, marcas, categorias y secciones de confianza.
- Catalogo de productos con filtros por categoria.
- Carrito persistente en localStorage con drawer y subtotales.
- Checkout por WhatsApp con resumen del pedido.
- Rutas de producto, politicas y contacto para operacion comercial.
